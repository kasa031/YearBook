# Changelog

All notable changes to the YearBook project will be documented in this file.

## [2.0.0] - 2025-11-04

### Added
- **Report System**: Users can now report posts that are inappropriate, spam, or contain incorrect information
- **Admin Panel**: Complete admin interface for reviewing and moderating reported content
- **Mobile Menu**: Hamburger menu for better mobile navigation
- **Keyboard Navigation**: Support for ESC key, Enter key, and other shortcuts
- **Loading States**: Better visual feedback during operations
- **Smooth Animations**: Enhanced UI with fade-in effects and transitions
- **Tag Icon**: Added icon for "Tag & Organize" feature card
